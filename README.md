# Little Angels Mabambani - School Management System

## Frontend Application

A modern, responsive school website built with React 19, Vite, and Tailwind CSS.

### Features

- 🎨 **Modern UI** - Built with Tailwind CSS and Framer Motion animations
- 📱 **Fully Responsive** - Works seamlessly on all devices
- 🔐 **Authentication Ready** - Firebase integration prepared
- 📊 **Dashboard Components** - Ready for student/teacher dashboards
- 📝 **Forms** - React Hook Form integration for seamless form handling
- 📄 **PDF Export** - jsPDF for generating reports
- 🎯 **SEO Optimized** - Meta tags and semantic HTML

### Tech Stack

**Frontend:**
- React 19
- Vite
- Tailwind CSS
- Framer Motion (animations)
- React Router (routing)
- React Hook Form (forms)
- Lucide React (icons)

**Backend (Prepared):**
- Firebase Authentication
- Firestore Database
- Firebase Storage
- Firebase Hosting

### Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/savagetoxic001/little-angels-mabambani.git
   cd little-angels-mabambani
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env
   # Update .env with your Firebase credentials
   ```

4. **Start development server**
   ```bash
   npm run dev
   ```

5. **Build for production**
   ```bash
   npm run build
   ```
